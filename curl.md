## curl в Linux

Скачать файл:  
`curl -L -O <url файла>`  
`-L` - используется для перенаправления (когда по ссылке на файл идет перенаправление)  
`-O` - сохраняет файл в текущий каталог с тем же именем  
`-o <путь/имя файла> - сохраняет файл в указанный католог с новым именем
